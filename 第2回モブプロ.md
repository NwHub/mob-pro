## PaizaCloudでssh-keyを生成（全員）
PaizaCloudからGitHubにpush出来るようにします。  
`ssh-keygen`の質問はエンター3回でOK
```
$ git config --global user.email "you@example.com"
$ git config --global user.name "Your Name"

$ ssh-keygen -t rsa -b 4096
$ cat ~/.ssh/id_rsa.pub
```
